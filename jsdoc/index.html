IMAT

The IMAT namespace holds all of the formal classes of the Intuit Mobile Automation Toolkit(IMAT).

IMAT.AssertionException

The IMAT.AssertionException object is thrown any time the a test failure is thrown due to an assertion API.

IMAT.BaseFunctionalTestSet

The IMAT.BaseFunctionalTestSet object is common test set object to be extended by all tests doing testing at the UI level.

IMAT.BaseTestSet

The IMAT.BaseTestSet is the most basic of all tests and does nothing more than provide stubbed out methods that the IMAT.TestRunner expects and initialize some basic instance variables related to its title etc.

IMAT.BaseView

The Base view is responsible for a few things.

IMAT.logLevels

Holds log level constants for the framework.

IMAT.platforms

Used to determine which platform to run against.

IMAT.settings

The IMAT.settings namespace should hold all things related to settings used while running tests.

IMAT.states

Holds state constants for the framework to be used when validating common states of a given view class.

IMAT.SuiteRunner

The IMAT.SuiteRunner object is the highest level object managing the test execution lifecycle.
