Namespace IMAT

The IMAT namespace holds all of the formal classes of the Intuit Mobile Automation Toolkit(IMAT). If an object is part of the IMAT namespace, then it should provide some sort of common functionality applicable to all types of consumers of this framework.
Defined in: IMAT.js.
